# Hangman Game (Python)

A text-based Hangman game built as part of my **100 Days of Code – Python** journey.

## Features
- Random word selection
- ASCII art for game stages
- Life tracking system
- Prevents repeated guesses (correct and incorrect)
- Clear win/lose conditions

## Project Structure
- `main.py` – game logic
- `hangman_words.py` – word list
- `hangman_art.py` – ASCII art and logo

## How to Run
1. Make sure all files are in the same folder
2. Run:
   ```bash
   python main.py
